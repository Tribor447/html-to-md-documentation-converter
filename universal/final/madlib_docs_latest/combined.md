MADlib Documentation

Apache MADlib is an open-source library for scalable in-database analytics. It provides data-parallel implementations of mathematical, statistical, graph and machine learning methods for structured and unstructured data.

Useful links:

- [MADlib web site](http://madlib.apache.org/)
- [MADlib wiki](https://cwiki.apache.org/confluence/display/MADLIB)
- [JIRAs for reporting bugs and reviewing backlog](https://issues.apache.org/jira/browse/MADLIB/)
- [User mailing list](https://mail-archives.apache.org/mod_mbox/madlib-user/)
- [Dev mailing list](https://mail-archives.apache.org/mod_mbox/madlib-dev/)
- User documentation for earlier releases: [v1.17.0](https://madlib.apache.org/docs/v1.17.0/index.html), [v1.16](https://madlib.apache.org/docs/v1.16/index.html), [v1.15.1](https://madlib.apache.org/docs/v1.15.1/index.html), [v1.15](https://madlib.apache.org/docs/v1.15/index.html), [v1.14](https://madlib.apache.org/docs/v1.14/index.html)
